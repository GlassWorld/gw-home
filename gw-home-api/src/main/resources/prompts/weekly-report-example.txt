예시 content 본문:

## 주요 작업 요약
- Chat-System 운영/권한/메시지 흐름 정비.
- HR-System 연계/세션/환경 구분 보정.
- 공통 문서와 구조 기준 정리.

## 상세 작업
### Chat-System
*UW*
- 사용자 권한 관리와 관리자/이용자 채팅 흐름 정리.
- UW 메인 채팅 화면, 매니저 화면, 통계 화면 구조 재정비.
- 관리자 콘텐츠 CRUD, 질문 상태 모델, 메시지 조회/응답 처리 구조 개선.
- 공지, 이벤트, 인사말, 질문유형 태깅 등 운영 기능 보강.

*리팩토링*
- UUID 기준 식별자 계약 정리와 대리접속 흐름 보정.

### HR-System
*연동/운영*
- 채팅 연계 점검과 SSO 연동 흐름 확인.
- 관리자 refresh 세션 적용과 빠른 이동/플로팅 메뉴 동작 보정.

*정비*
- 환경 구분, API 응답 필드, 클라이언트 타입 오류 정리.
- 사이드 메뉴 상태 유지와 화면 사용성 개선.

### 공통 문서/구조 정비
*문서정리*
- AI 문서 구조, 라우팅, work 기준 정리.
- 작업 규칙, 커밋 규칙, 템플릿과 아카이브 정리.

*구조정비*
- 프론트 feature 구조와 mapper/adm 패턴 기준 반영.
