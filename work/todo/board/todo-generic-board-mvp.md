# 요청 작업 계획 문서

## Goal

- 목적:
  - 기존 `board/comment/file` 구현을 확장해 실제 사용 가능한 게시판 MVP를 만든다.

## Scope

- 포함:
  - 게시글 목록 검색 조건 확장
  - 게시글 작성/수정 화면 구현
  - 마크다운 입력 및 미리보기
  - 게시글 상세의 마크다운 렌더링
  - 댓글/답글 작성, 수정, 삭제 UI 연결
  - 게시글 첨부파일 연결과 다운로드 기본 흐름
  - 카테고리 조회 API 추가
- 제외:
  - 게시판 마스터 테이블 신설
  - 게시판별 세부 권한 설정 UI
  - 좋아요, 북마크, 태그 고도화
  - 조회수 중복 증가 방지 고도화

## Current State

- 현재 상태:
  - 게시글 CRUD API와 댓글 API는 존재한다.
  - 프론트는 목록/상세 기본 페이지만 있고 작성/수정/댓글/파일 UX가 비어 있다.
  - 파일 업로드는 독립 기능만 있고 게시글 첨부 구조는 없다.
- 제약:
  - 프로젝트 규칙상 MyBatis만 사용해야 한다.
  - 외부 식별자는 `uuid`만 노출해야 한다.
  - 기존 구조와 충돌을 최소화해야 한다.

## Action Items

1. 게시판 백엔드에 카테고리 조회, 검색 조건 확장, 첨부파일 연결 모델을 추가한다.
2. 댓글 서비스에 답글 depth 제한과 화면 연동에 필요한 응답 흐름을 정리한다.
3. 게시판 프론트에 작성/수정 폼, 마크다운 미리보기, 파일 업로드, 댓글 트리를 구현한다.
4. 가능한 테스트와 타입 검증을 실행해 회귀를 확인한다.

## Done Criteria

- 게시글 작성/수정이 실제 API와 연결된다.
- 상세 화면에서 마크다운 본문이 sanitize 되어 렌더링된다.
- 댓글과 답글을 화면에서 작성, 수정, 삭제할 수 있다.
- 첨부파일이 게시글과 연결되고 상세 화면에서 다운로드할 수 있다.
- 목록에서 검색 조건과 기본 메타 정보가 노출된다.
