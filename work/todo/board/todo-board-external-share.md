# 게시판 외부 공유 기능 계획

## Goal

- 목적: 게시글을 로그인 없이 기간 제한 링크로 조회할 수 있는 읽기 전용 외부 공유 기능을 구현한다.

## Scope

- 포함: 공유 토큰 발급/해제/재발급, 만료 기간 설정, 선택 비밀번호, 공유 상태 조회, 외부 전용 조회 API, 외부 전용 페이지, 내부 상세의 공유 설정 UI
- 제외: 외부 협업 기능, 외부 댓글 작성, 외부 좋아요, 내부 상세 화면 재사용, 무기한 공개

## Current State

- 현재 상태: 게시글은 일반 상세 API와 내부 상세 페이지로만 조회되며, 외부 공유 전용 구조는 없다.
- 제약: 최대 만료 기간은 30일, 공유는 항상 읽기 전용, 세션이 있어도 공개 페이지에서는 내부 권한을 적용하지 않아야 한다.

## Action Items

1. 공유 저장용 DDL, VO/JVO, Mapper, 정책 상수를 추가한다.
2. 공유 관리 API와 공개 조회 API를 구현하고 보안 경로를 설정한다.
3. 게시글 상세에 공유 설정 모달과 상태 표시를 추가한다.
4. 외부 공유 전용 페이지와 타입/컴포저블을 구현한다.
5. 테스트와 검증 코드를 추가하고 핵심 흐름을 확인한다.

## Done Criteria

- 로그인 없이 공유 링크로 게시글 읽기 전용 조회가 가능하다.
- 공유 해제와 재발급이 즉시 반영되고 기존 링크는 무효화된다.
- 만료, 해제, 비밀번호 오류, 삭제된 게시글 상태가 구분되어 안내된다.
- 외부 공유 페이지에서는 수정, 삭제, 댓글 작성 등 어떤 상호작용도 제공되지 않는다.
